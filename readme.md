## git command
1. ターミナルを出す
2. 以下のコマンドを打つ
   1. $はターミナルに入力しないこと
   2. コマンドはターミナルに入力する
   3. $以外の部分をコピペすること
   4. git commitのコマンドは""の中に作業内容を記載すること
```
$ cd C:\xampp\htdocs\sbar-system-na
$ git add .
$ git commit -m "1205 宿題完了‼"
$ git push
```
